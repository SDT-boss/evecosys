    2. `export interface DbConnectionParams { engine: BYODBEngine; host: string; port: number; database: string; user: string; password: string; ssl?: boolean }`.

    7. `export function normalizeCredential(input: BYODBCredentialInput): DbConnectionParams` — for `kind: 'structured'` validate every field of `params` is present (non-empty host/database/user/password, port a positive integer, engine in BYODB_ENGINES) and return it; for `kind: 'connectionString'` parse via `new URL(connectionString)` and map: protocol prefix `postgres`/`postgresql` → engine 'postgres', `mysql` → engine 'mysql' (must match `input.engine`, else throw CredentialValidationError), `url.hostname` → host, `Number(url.port)` (default 5432 for postgres, 3306 for mysql when empty) → port, `url.pathname.replace(/^\//,'')` → database, `decodeURIComponent(url.username)` → user, `decodeURIComponent(url.password)` → password. Throw `CredentialValidationError` with a message that does NOT include the password value on any invalid/missing field.

    Do NOT add the `pg` or `mysql2` packages in this file — this module is pure parsing/validation with zero runtime DB dependency. (Drivers are introduced in Plan 02 behind the probe interface.)

       `GRANT EXECUTE ON FUNCTION public.delete_byodb_secret(uuid) TO service_role;`
